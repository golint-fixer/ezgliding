THIS DOCUMENT DESCRIBES THE FORMAT OF THE TURNING POINT, 
AIRPORT, AND OUTLANDING FIELD DATABASE WELT2000. IT DOES NOT 
COVER THE MAP FILE FORMAT OF WELT2000.MAP.

THE DOLLAR SIGN ($) AS FIRST CHARACTER MEANS: IGNORE LINE    
COLUMN 1-6:    SHORTFORM FOR GPS INPUT 6 CHARACTERS          
ALL THE NAMES ARE UNIQUE IN COLUMN 1-6                       
NAMES FOR AIRFIELDS ARE 5 CHARS LONG YOU CAN DELETE THE '1'  
VERY UNCLEAR AIRSTRIPS HAVE NUMBER '2' IN COL 5 NOT SEEN     
    BY HTTP://EARTH.GOOGLE.COM   
ALL OUTLANDING FIELDS HAVE NUMBER '2' IN COL 6               
COLUMN 8-19  SHORTFORM FOR ZANDER 940                        
IF COL 7 NOT EMPTY: TAKE THIS CHAR AS CHAR 12 FOR ZANDER

    SPECIAL REMARKS TO AIRFIELDS AND OUTLANDING FIELDS:      
 1. IF THE ICAO IDENT IS AVAILABEL, THIS CODE IS DIRECTLY    
    FOLLOWED THE '*' (EXPL FOR BAYREUTH "EDQD")              
    IN CASE THE # INDICATOR IS FOLLOWED BY A BLANK NO        
    ICAO CODE IS AVAIBLE. IN THESE CASES YOU CAN TAKE COL    
    1-4 FOR ID: THEY ARE NEVER IDENTICAL WITH ANY ICAO ID.   
 2. ULTRALIGHT FIELDS ARE LIKE OUTLANDING FIELDS ("*ULM")    
 3. OUTLANDINGFIELDS FROM DIFFERENT CATALOGUES ARE LABELED   
    *FLXX STANDS FOR EMERGENCY CATALOGUE NUMBER              
    XX IS THE NUMBER OF THE FIELD WITHIN A CATALOGUE         
 4. "#GLD" AT COL 24:27 OR "GLD#" AT 21:24 MEANS GLIDERSITE  
 COL29: A=ASPH C=CONC L=LOAM S=SAND Y=CLAY G=GRAS V=GRAVEL   
 COL29: D=DIRT                                               
     30 - 32 LENGTH OF RUNWAY IN METERS TO MULTIPLY BY 10    
     33 - 34  AND 35-36 DIRECTION OF RUNWAYS:                
                  INCLUDES SOME FURTHER INFORMATION:         
               A: 08/26 MEANS THAT THERE IS ONLY ONE RUNWAYS 
                  08 AND (26=08 + 18)                        
               B: 17/07 MEANS THAT THERE ARE TWO RUNWAYS,    
                        BUT 17 IS THE MAIN RWY SURFACE LENGTH
              C: IF BOTH DIRECTIONS ARE IDENTICAL (04/04)    
                  THIS DIRECTION IS STRONGLY RECOMMENDED     
        37 - 41   FREQUENCY THE BEST FOR GLIDERS             
        41 - 41   12337 BECOMES 123.375, 13102 IS 131.025    
    COL 42 - 45       ELEVATION IN METER                     
    COL 42 - 42  0 =  ELEVATION IN METER NOT PROVED
